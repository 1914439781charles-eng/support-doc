# Description

This article provides instructions on how to customize and create a dashboard in Datacake.

# Requirement

- Milesight sensor
- Datacake platform

# Configuration

Step1: Add data fields in Datacake.

Step2: Create widgets in your dashboard. 

Step3: Bind widget to the corresponding fields.

After successfully creating the dashboard and binding the corresponding fields, the data reported by the sensor will be updated in real time on the dashboard.
